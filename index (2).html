Fore Group Onboarding Hub

Last updated: May 2026● Live

Single source of truth for all client onboardings at Fore Group. Deals flow in automatically from HubSpot and everything lives here — checklists, progress, documentation and Claude actions.

How it works

1
Deal closes in HubSpot
A new deal hitting Closed-Won automatically appears in the Inbox for Tiaan to review.
2
Tiaan reviews & assigns
Check the products, dates and details pulled from HubSpot. Assign to a specialist and approve.
3
Specialist works the checklist
Click any row to open the checklist. Mark steps done as you go. Claude action buttons trigger browser automation.
4
Hub writes back to HubSpot
Specialist assignments and go-live date changes sync back to HubSpot automatically.

HubSpot sync

Last updated: May 2026

What flows from HubSpot → Hub (once, on import)

  • Client name (deal name)
  • HubSpot Deal ID
  • Expected Go-Live Date (deal property)
  • Requested Implementation Start Date (onboarding ticket)
  • Line items → mapped to product tags

Product line item mapping

Lightspeed Golf → Golf
Lightspeed Retail (R) → Retail (R-Series)
Lightspeed Retail (K) → Retail (K-Series)
Lightspeed Restaurant K-Series → Restaurant (K-Series)
Lightspeed Restaurant (L) → Restaurant (L-Series)
Ecosystem Support → Ecosystem Support
Custom App → Custom App

What flows from Hub → HubSpot (live)

  • Specialist assignment → updates Ticket Owner
  • Go-live date changes → updates Expected Go-Live Date on the deal

Go-live date is owned by the hub after import. Never change it in HubSpot directly — change it in the hub and it will sync back.

Syncing

Click the "Sync HubSpot" button in the Inbox to check for new Closed-Won deals. The hub checks automatically each time it loads.

Step 1: Provisioning

Last updated: May 2026Golf

What to do

  • Access the provisioning form, enter Deal & Golf Club Name and your contact details
  • Select Account Provisioning, complete Course Setup, click Submit
  • Credentials arrive via Slack — store immediately in 1Password

1Password naming

[Club Name] - Lightspeed Golf
[Club Name] - Lightspeed Retail
[Club Name] - Lightspeed Restaurant

After receiving credentials

  • Update admin username to Fore Group (except L-Series)
  • For K-Series: gather Email, Trade Name, Contact Name, Phone and Address — forward to Medelyn at Lightspeed

Copy the website URL from an existing 1Password entry so 1Password recognises the login correctly.

Step 3: Welcome email

Last updated: May 2026Golf

After the sales rep's introduction, send the Welcome Email using the relevant HubSpot template.

  • Select template matching the client's tech stack
  • Include Questionnaire link with Deal ID appended: ?hubspot=[DEAL_ID]
  • Attach Member, Retail and Restaurant (K-Series) Import Files if applicable
  • Include Custom App Request Form if relevant
  • Include Website Request Form if client has a Custom Website
  • Include Hardware Recommendations — remove irrelevant sections

Last updated: May 2026Golf

Best practices

Last updated: May 2026
  • Never assume — schedule a check-in rather than guessing
  • Go-live date — always change in the hub, never in HubSpot directly
  • Communicate proactively — flag blockers early
  • Keep the hub up to date — mark steps done as soon as they're complete
  • Credentials in 1Password only
  • Split training — Golf, Retail and F&B separately; managers separate from staff
